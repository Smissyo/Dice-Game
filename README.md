# Dicee Challenge 🎲

A simple two-player dice game built with HTML, CSS, and JavaScript. 

[👉 Click here to play the live game!](https://smissyo.github.io/Dice-Game/))

## Features
* Generates a random dice roll for two players upon page refresh.
* Dynamically updates the images and declares a winner (Player 1, Player 2, or a Draw).
* Built using core JavaScript DOM manipulation techniques (`querySelector`).
